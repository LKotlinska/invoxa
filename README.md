# Invoxa

Invoxa is an invoice management app built with **Angular 22** and **Angular Material** currently under development. It lets you create, edit, and track invoices, manage customers and products, and keep everything backed by a **Supabase** database, with **RxJS** used throughout for reactive data flow.
